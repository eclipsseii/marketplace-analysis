# Marketplace Analysis

## Project Goal

Analyze customer behavior, orders, and revenue metrics using a real e-commerce marketplace dataset.

## Dataset

Brazilian E-Commerce Public Dataset by Olist.

## Tools

- PostgreSQL
- SQL
- TablePlus
- GitHub
